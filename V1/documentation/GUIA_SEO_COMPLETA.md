## SEO TÉCNICO

### Sitemap XML

**Estructura:**
```
sitemap.xml
├── sitemap-es.xml (Español)
├── sitemap-en.xml (Inglés)
└── sitemap-news.xml (Artículos Blog)
```

**Ubicación:** /sitemap.xml
**Incluir:** Todas las páginas + artículos blog

### Robots.txt

```
User-agent: *
Allow: /
Disallow: /wp-admin/
Disallow: /wp-includes/
Disallow: /portal-de-padres/
Disallow: /private/

Sitemap: https://valderramainternationalschool.com/sitemap.xml
```

### Velocidad de Página

**Objetivos:**
- Tiempo de carga: < 2 segundos (mobile < 3 segundos)
- Largest Contentful Paint (LCP): < 2.5 segundos
- First Input Delay (FID): < 100ms
- Cumulative Layout Shift (CLS): < 0.1

**Recomendaciones:**
- Usar WP Rocket para caching
- Optimizar imágenes (WebP format)
- Minificar CSS/JS
- Usar CDN (Cloudflare)
- Lazy loading en imágenes

### Mobile-Friendly

- Diseño responsive ✅ (ya implementado)
- Touch-friendly buttons (mín 48px)
- Legibilidad en mobile
- No usar Flash
- Viewport meta tag configurado

## ANALYTICS Y MONITOREO

### KPIs a Rastrear

**Google Analytics 4:**
```
1. Sesiones orgánicas mensuales
2. Tasa de clics (CTR) desde Google
3. Páginas por sesión
4. Duración promedio de sesión
5. Tasa de conversión (formularios contacto)
6. Posición promedio en Google Search Console
```

### Google Search Console

**Configurar:**
- Verificar propiedad (DNS o etiqueta HTML)
- Enviar sitemap.xml
- Monitorear palabras clave
- Revisar errores de indexación
- Analizar clics y impresiones

### Herramientas Recomendadas

- Google Analytics 4 (gratis)
- Google Search Console (gratis)
- Yoast SEO Plugin (freemium)
- Semrush o Ahrefs (premium)
- Ubersuggest (freemium)
